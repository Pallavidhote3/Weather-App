
# 🌤️ Weather App

A simple weather app made using HTML, CSS & JavaScript. It fetches real-time weather data from the OpenWeatherMap API.

## 🔧 Technologies Used
- HTML
- CSS (Flexbox)
- JavaScript (Fetch API)
- OpenWeatherMap API
